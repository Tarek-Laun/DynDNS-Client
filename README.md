# DynDNS-Client
A NodeJS DynDNS Client.

## Config 
```json
{
    "DNSs": [{
        "Service": "Strato",
        "Hostname": "domain.net",
        "Domain": "subdomain.domain.net",
        "Password": "Password"
    }],
    "IpCheckInterval": 1 
}
```

## Setup
```bash
npm i
node main.js
```
